karam的使用

Posted by xiaozhou on 2020-07-18

本篇教程介绍如何将数据库的数据导入Karam，生成RDF

注意：如果不连VPN，图片可能显示不出来。

1. 下载Karam，启动并用浏览器打开Karam的页面

Karam官网

Karma安装教程

2. 点击页面顶部的Import，选择数据库导入，填入连接数据库的信息，导入要处理的表

由于我是把数据存放在数据库中，因此我选择数据库导入数据，你也可以选择文件导入的方法。

注意事项：确保已经启动数据库；如果你是导入json，csv，xml等类型的数据，要将文件放在/karma/user-uploaded-files中。

下图是我karam的目录

3. 导入这个表的Ontology，可以使用Protege来构建本体模型

你需要在Protege中设置表的Class，Object properties，Data properties。

Object properties表示Class之间的关系，Data properties表示Class的属性；数据属性相当于树的叶子节点，只有入度，而没有出度；然后保存文件。

Protege下载链接

4. 然后在Karam中设置每一列Semantic Types，Class之间的Linking

注意：在设置某一列的Semantic Types时要对应本体模型的Class和properties。

下图是我构建好的一个例子。

5. 构建完成后点击Publish导出RDF和R2RML Model

导出的R2RML文件需要放在/karma/user-uploaded-files中，下次打开该表时点击

Apply R2RML Model -> from file应用这个模型，就不需要再设置列的Semantic Types和Class的Linking。
